Some sections of this book were originally written by Dr. Brad Miller as Java for Python Programmers. We would like to extend a special note of gratitude to Jesse Walker-Schadler 2  who had the vision to create the CTurtle library 3  which makes graphing in C++ particularly easy for Python programmers because it used the Python Turtle library syntax, and for his excellent work in porting it to Runestone.
